# djaynowplaying
A project that can retrieve the currently playing track information from djay (a DJ software) and display it on a local web page.
可以把DJay正在播放的音乐信息显示在本地服务器网页上。

Usage:

Download the release version.
Follow the instructions to configure the djay database path and other parameters.
Open the local server address.
Note:
The release version is for Windows only. The web page template can be customized. Simply open the local server address in the software you want to use (such as OBS).

使用方法：
1.下载发布版。
2.根据提示设置好djay数据库路径和其它参数。
3.打开本地服务器地址。 
注意：发布版只适用于windows，网页模板可更改，在需要使用的软件中（如OBS）打开本地服务器地址即可。
